## ğŸ“– Project Overview

This project is an **Arduino UNO-based stepper motor control system** that allows the user to **control either the speed or the angle** of a stepper motor using a **rotary encoder** and an **LCD keypad shield**.

- The **rotary encoder** is used to adjust values digit by digit.  
- The **LCD Keypad Shield** allows switching between options, selecting modes, and navigating digits.  
- The **stepper motor driver** is controlled by pulse signals generated by the Arduino.

### âœ… Features
1. **Two control modes:** Speed control or Angle control.
2. **Switch between modes** using LEFT/RIGHT buttons and SELECT to confirm.
3. **Digit-by-digit adjustment** for speed (in RPM) or angle (in degrees).
4. **Clockwise/Counterclockwise direction** toggle.
5. **Return to home page:** Hold SELECT for 5 seconds to switch between modes.

## âš™ï¸ How to Use

1. **Start the system** â€“ the home page displays two options:  
   - **Speed Control**  
   - **Angle Control**

2. **Select Mode:**
   - Use **LEFT/RIGHT** to move between modes.  
   - Press **SELECT** to enter the chosen mode.

3. **Adjust values:**
   - The display shows the value with **digits** (e.g., `000.0 CC`).  
   - Move the cursor using **LEFT/RIGHT** to select a digit.  
   - Increase or decrease the value using **UP/DOWN**.  
   - Toggle direction (Clockwise / Counterclockwise) by moving cursor to `CC` and pressing **UP/DOWN**.

4. **Switching modes:**
   - Hold **SELECT** for 5 seconds to return to the **home page**.  
   - Choose the other mode (Speed â†” Angle) and repeat adjustments.
